# 🚀 Welcome to Lagos

**Lagos** is a modern, feature-rich **Bootstrap 5 Admin Dashboard Template** designed to help developers build responsive, scalable, and high-performance web applications with confidence. Built with **Bootstrap 5**, **SASS**, and **jQuery**, Lagos combines clean architecture, reusable components, and an intuitive user interface to accelerate development.

Whether you're creating an **Admin Dashboard**, **CRM**, **SaaS Platform**, **Analytics Dashboard**, **Project Management System**, or any business-focused application, Lagos provides everything you need to build professional interfaces quickly and efficiently.

With an extensive collection of ready-to-use pages, flexible layouts, reusable UI components, responsive data tables, authentication screens, and production-ready features, Lagos enables developers to focus on building applications instead of creating the foundation from scratch.

### 📦 Setup Instructions

#### **1. Download or Clone Lagos**

Get the latest version of the **Lagos** template by downloading the project archive or cloning the repository to your local environment.

#### **2. Install Project Packages**

Navigate to the project directory and install all required dependencies using the following command:

```bash
npm install
```

#### **3. Launch the Development Environment**

After the installation is complete, start the development server to compile assets and preview the template in your browser:

```bash
gulp
```

#### **4. Start Building**

You're all set! Customize layouts, switch between **Light**, **Dark**, **RTL**, and **Boxed** modes, explore the UI components, data tables, animations, authentication screens, and error pages, then begin developing your dashboard with Lagos.
